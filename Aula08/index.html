GitHub

Para repositório novo:

  1. git config --global user.name "username"
  2. git config --global user.email "user@email.com"
  3. git init
  4. git add [. (para todos os arquivos)] ou [nomeDoArquivo.exemplo]
  5. git commit -m "mensagem do arquivo"
  6. git branch -M main (Branch-Name)
  7. git remote add origin "https://https://github.com/LinkDoCaminho" (Remote-Name)
  8. git push -u origin main

Para repositório clonado:

  1. git config --global user.name "username"
  2. git config --global user.email "user@email.com"
  3. git init
  4. git add [. (para todos os arquivos)] ou [nomeDoArquivo.exemplo]
  5. git commit -m "mensagem do arquivo"
  6. git push -u origin main
